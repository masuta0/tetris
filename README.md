# テトリス - Tetris Game

高性能で美しい日本語対応のテトリスゲームです。モダンなUI、複数のテーマ、難易度設定、ランキングシステムなど、充実した機能を実装しています。

## 🎮 主な機能

### 基本的なゲームプレイ
- 🕹️ クラシックなテトリスゲームプレイ
- 🎯 スコア、レベル、ライン数の表示
- 🔄 ネクストピース、ホールド機能
- 👻 ゴーストピース機能
- ⏸️ ポーズ機能

### 高度なゲームシステム
- 🛡️ 壁際回転（Wall Kick）システム
- ⚡ ソフトドロップ＆ハードドロップ
- 🕐 接地猶予時間（500ms）
- 🚨 危険ゾーン警告システム
- 💫 スペシャルエフェクト（4列同時消去時）

### カスタマイズ機能
- 🎨 6種類の美しいテーマ
  - ダークモード（デフォルト）
  - ライトモード
  - ネオン
  - レトロ
  - ナチュラル
  - オーシャン
- 🎚️ 3段階の難易度設定
  - イージー（1200ms）
  - ノーマル（1000ms）
  - ハード（700ms）

### データ管理
- 💾 ローカルストレージ対応
- 🏆 ハイスコア保存
- ⏱️ 総プレイ時間記録
- 📊 総ライン数記録
- 👤 プレイヤー名設定

### ランキングシステム
- 🥇 ローカルランキング（上位10件）
- 📝 プレイヤー名付きスコア記録
- 📈 スコアに応じた順位付け

## 🎯 遊び方

### 基本操作
- **←→** 左右移動
- **↑** 回転
- **↓** ソフトドロップ（加速落下）
- **スペース** ハードドロップ（即着地）
- **Ctrl** ホールド（ピース保持）
- **P** ポーズ

### ゲームの流れ
1. ホーム画面から「ソロプレイ」を選択
2. 難易度を選択（イージー、ノーマル、ハード）
3. 「ゲーム開始」ボタンで開始
4. ラインを消してスコアを稼ぐ
5. ゲームオーバー後はリスタートまたはホームに戻る

## 🛠️ 技術的な詳細

### 使用技術
- **HTML5** - セマンティックな構造
- **CSS3** - モダンなスタイリング、グリッド、フレックスボックス
- **JavaScript (ES6+)** - クラスベースの構造
- **Canvas API** - ゲームレンダリング
- **LocalStorage** - データ永続化

### ゲームアルゴリズム
- **衝突判定** - AABB（Axis-Aligned Bounding Box）
- **回転システム** - Super Rotation System（SRS）ベース
- **壁際補正** - 複数のオフセットパターンによる壁際回転
- **ライン消去** - 完全な行の検出と削除

### パフォーマンス最適化
- **効率的なレンダリング** - 必要な部分のみ再描画
- **メモリ管理** - オブジェクトの再利用
- **アニメーション最適化** - requestAnimationFrame使用

```css

    background: linear-gradient(135deg, #your-color1 0%, #your-color2 100%);
    color: #your-text-color;
}
```

### 難易度の調整
```javascript
// game.js内のsetDifficultyメソッドを編集
setDifficulty(difficulty) {
    switch (difficulty) {
        case 'easy':
            this.dropInterval = 1200; // 落下間隔（ms）
            break;
        // 他の難易度...
    }
}
```

### ランキングシステムの拡張
```javascript
// LocalRankingクラスをカスタマイズ
class LocalRanking {
    constructor() {
        this.maxRankings = 10; // 表示件数を変更
        // 他の設定...
    }
}
```
